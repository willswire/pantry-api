# Endpoints

## User
* [ ] `POST/user`
   * Create new user
* [ ] `GET/user`
    * Return all users
* [ ] `GET/user/${id}`
    * Return individual user
* [ ] `PUT/user/${id}`
    * Update individual user information
* [ ] `GET/user/${id}/lists`
    * Return individual user's list

## Item
* [ ] `POST/item`
   * Create new item
* [ ] `GET/item`
    * Return all items
* [ ] `GET/item/${id}`
    * Return individual item
* [ ] `DELETE/item/${id}`
    * Delete individual item

## List
* [ ] `POST/list`
   * Create new list
* [ ] `GET/list/${id}`
    * Return individual list
* [ ] `DELETE/list/${id}`
    * Delete individual list
