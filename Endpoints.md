# Endpoints

## User
* [x] `POST/users`
   * Create new user
* [x] `GET/users`
    * Return all users
* [x] `GET/users/${id}`
    * Return individual user
* [x] `PUT/users/${id}`
    * Update individual user information
* [x] `GET/users/${id}/lists`
    * Return individual user's lists

## Item
* [x] `POST/items`
   * Create new item
* [x] `GET/items`
    * Return all items
* [x] `GET/items/${id}`
    * Return individual item
* [x] `DELETE/items/${id}`
    * Delete individual item

## List
* [x] `POST/lists`
   * Create new list
* [x] `GET/lists/${id}`
    * Return individual list
* [x] `DELETE/lists/${id}`
    * Delete individual list
